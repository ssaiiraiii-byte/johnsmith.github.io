2. 盈利能力对比分析

为确定最优运行策略，本报告将分别对“两充两放”与“一充一放”两种方案进行详细的经济性测算，并对关键指标进行对比。

2.1 方案一：“两充两放”策略

该策略下，储能系统每日执行两次完整的充放电循环，分别利用“峰谷价差”和“尖峰平价差”进行套利，以实现收益最大化。

2.1.1 现金流测算

第一步：计算年总收益

年总收益 = 峰谷套利收益 + 尖峰平价套利收益 + 需量管理收益 + 需求侧响应收益

单次有效放电量 = 额定容量 × DOD × 转换效率 = 4076 kWh × 90% × 90% = 3301.56 kWh

• 冬夏季(120天)套利收益 = 120天 × (0.6425+0.6178)元/kWh × 3301.56 kWh = 50.18 万元

• 非冬夏季(210天)套利收益 = 210天 × (0.6240+0.4942)元/kWh × 3301.56 kWh = 77.28 万元

• 年度套利总收益 = 50.18 + 77.28 = 127.46 万元

需量管理年收益 = 50.4 万元 (来源: 案例表2)

需求侧响应年收益 = 1 万元 (来源: 案例表2)

年总收益 = 127.46 + 50.4 + 1 = 178.86 万元

第二步：计算年总成本与税费

年运营成本 = 管理费 + 其他费用 + 保险费 + 税金附加

• 管理费: 13.00万元/年; 其他费用: 4.08万元/年; 保险费: 0.04万元/年 (400元)。

• 税金附加: 1-5年为0.04万元/年, 6-10年为0.03万元/年。

• 年运营成本 (1-5年) = 13 + 4.08 + 0.04 + 0.04 = 17.16 万元

• 年运营成本 (6-10年) = 13 + 4.08 + 0.04 + 0.03 = 17.15 万元

年折旧 = (初始投资 - 预计残值) / 折旧年限 = (611.40 - 611.40 × 14%) / 10 = 52.58 万元

年等额本息还款 = 贷款额 × [i(1+i)ⁿ] / [(1+i)ⁿ-1] = 427.98 × [4.65%(1.0465)¹⁰] / [(1.0465)¹⁰-1] = 54.47 万元

所得税：根据“三免三减半”政策（1-3年税率0%，4-6年12.5%，7-10年25%）计算。应纳税所得额 = 收入 - 运营成本 - 折旧 - 当期利息。

2.1.2 净现值 (NPV) 计算

NPV 计算公式:
NPV =
∑ n t=0
NCFt (1 + r)t

使用折现率 r = 5%，对上表现金流进行折现加总，计算得出：

NPV = 604.08 万元

结论：NPV (604.08万元) 远大于0，表明项目投资的预期收益超过了资本成本，具备很强的财务可行性。

2.1.3 内部收益率 (IRR) 计算

IRR 定义: 使NPV等于0的折现率
∑ n t=0
NCFt (1 + IRR)t
= 0

通过插值法或财务函数，找到使上述现金流净现值为0的折现率。

IRR = 33.68%

结论：IRR (33.68%) 远高于基准折现率 (5%)，表明项目具有极强的盈利能力和抗风险能力。

2.1.4 平准化度电成本 (LCOE) 计算

LCOE 计算公式:
LCOE =
∑ n t=0
投资t + 运营成本t - 残值t (1 + r)t
∑ n t=1
年放电量t (1 + r)t

分子 (总成本现值):

总成本现值 = 初始投资 + Σ(年运营成本现值) - 残值现值

• 运营成本现值 = Σ [年运营成本 / (1+5%)ᵗ] = 132.47 万元

• 残值现值 = (611.40 × 14%) / (1+5%)¹⁰ = 52.55 万元

• 总成本现值 = 611.40 + 132.47 - 52.55 = 691.32 万元

分母 (总放电量现值):

年放电量 = 3301.56 kWh/次 × 2次/天 × 330天 = 2,179,030 kWh

总放电量现值 = Σ [年放电量 / (1+5%)ᵗ] = 2,179,030 × PVIFA(5%, 10年) = 2,179,030 × 7.7217 = 16,824,838 kWh

LCOE = 6,913,200 元 / 16,824,838 kWh = 0.411 元/kWh

结论：LCOE (0.411元/kWh) 显著低于项目的主要套利价差，表明项目在单位发电成本上具有很强的经济性。

2.2 方案二：“一充一放”策略

该策略下，储能系统每日仅执行一次充放电循环，利用“峰谷价差”进行套利。此方案循环次数少，对电池寿命更友好，但可能牺牲部分收益。

2.2.1 现金流测算

第一步：计算年总收益

年总收益 = 峰谷套利收益 + 需量管理收益 + 需求侧响应收益

单次有效放电量 = 3301.56 kWh (同方案一)

年套利收益 = [120天 × 0.6425元/kWh + 210天 × 0.6240元/kWh] × 3301.56 kWh = 68.72 万元

需量管理与需求侧响应收益不变，分别为 50.4 万元和 1 万元。

年总收益 = 68.72 + 50.4 + 1 = 120.12 万元

年运营成本、折旧、贷款本息与方案一相同，仅收入和所得税发生变化。计算得出全生命周期的股权净现金流如下：

2.2.2 净现值 (NPV) 计算

使用折现率 r = 5%，对“一充一放”策略的现金流进行折现加总。

NPV = 102.97 万元

结论：NPV (102.97万元) 大于0，项目在此策略下同样具备财务可行性，但价值创造能力显著低于“两充两放”策略。

2.2.3 内部收益率 (IRR) 计算

计算使“一充一放”策略现金流净现值为0的折现率。

IRR = 18.04%

结论：IRR (18.04%) 仍远高于基准折现率 (5%)，表明项目盈利能力良好，但与“两充两放”策略的33.68%相比，回报率较低。

2.2.4 平准化度电成本 (LCOE) 计算

分子 (总成本现值): 与方案一相同，为 691.32 万元。

分母 (总放电量现值):

年放电量 = 3301.56 kWh/次 × 1次/天 × 330天 = 1,089,515 kWh

总放电量现值 = 1,089,515 × PVIFA(5%, 10年) = 1,089,515 × 7.7217 = 8,412,419 kWh

LCOE = 6,913,200 元 / 8,412,419 kWh = 0.822 元/kWh

结论：LCOE (0.822元/kWh) 显著高于“两充两放”策略。这是因为总成本在更少的放电量上进行分摊，导致单位成本上升。

最终结论与建议

通过对两种运行策略的详细财务测算与对比，可以得出以下结论：

  • 经济性：两种方案在财务上均为可行（NPV > 0, IRR > 5%）。然而，“两充两放”策略的盈利能力远超“一充一放”策略，其净现值高出近5倍，内部收益率高出超过15个百分点。
  • 成本效益：“两充两放”策略通过更高效地利用储能设备，将总成本分摊到更多的放电量上，使得其平准化度电成本（LCOE）仅为“一充一放”策略的一半，成本优势极为明显。
  • 风险考量：虽然“两充两放”策略每日循环次数更多，理论上对电池寿命的耗损更大，但其带来的巨大经济效益足以覆盖潜在的、通过合理运维可以减缓的额外衰减成本。

综上所述，尽管需要考虑电池长期衰减的管理，但从纯粹的经济性角度出发，A公司应毫无疑问地选择“两充两放”作为其工业储能项目的运行策略。该策略能为公司创造最大化的经济价值，是实现项目投资回报最优解的必然选择。
